# Pong Game

A fully funcional, object-oriented recreation of the classic 1972 Pong arcade game. Built using Python's Turtle graphics module.

## Features
* **Two-Player Local Multiplayer:** Independent controls for both paddles.
* **Physics & Collision Engine:** Calculated deflections for top/bottom walls and left/right paddles.
* **State Management:** Dynamic scoreboard that tracks points and automatically resets the ball to the center after a goal.
* **Boundary Detection:** Programmed limits to prevent players from moving off the screen.

## Controls
* **Right Player:** `Up` and `Down` arrow keys
* **Left Player:** `W` and `S` keys

## How to Run
1. Install Python
2. Clone this repository.
3. Run:
   ```bash
   python main.py
   ```
